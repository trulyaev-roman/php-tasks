Основные команды GIT:
==============================================================

clone - получение изначального репозитория
branch :
  -m: переименование ветки
  - просмотр списка веток
  -d: удаление слитой в master ветки
  -D: удаление ветки

checkout :
  - переключение на ветку
  -b: создание ветки

rebase : - перемотка

merge : - слияние

pull : получение данных и слияние из удалённого репозитория
fetch : получение данных
push : отправка данных на удалённый репозиторий

cherry-pick : перенос определённых коммитов

reset: - откат изменений до нужного коммита

remote:
  -add: - добавить адрес и имя удалённого репозитория
  -rm: - удалить адрес и имя удалённого репозитория
  -v: - получить информацию о текущих подключённых репозиториях

stash:
  - : спрятать изменения
  apply: показать спрятанные изменения
